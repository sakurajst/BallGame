# 项目简介

本项目计划开发一个Android平台的战斗冒险类游戏BallGame，以红球、灰球、黑球分别代表正能量、普通能量、负能量，以及它们之间的相互作用来模拟现实生活中人们受周围同伴情绪感染这一现象。
